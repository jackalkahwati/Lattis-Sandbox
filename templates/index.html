How to Use the Lattis API Sandbox

  1. Select an API endpoint from the dropdown in the sidebar.
  2. The Request URL and Request Body will be populated automatically.
  3. Modify the Request Body if needed for POST or PUT requests.
  4. Click the "Send Request" button to make the API call.
  5. The response will be displayed in the Response section.
  6. Use the map to visualize vehicle locations and interact with specific locations.
  7. Enter queries in the input box above the map for natural language interactions.

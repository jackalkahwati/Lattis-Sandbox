How to Use the Lattis API Sandbox

  1. Enter your API access token or generate a test token.
  2. Select an API endpoint from the list or use the search feature.
  3. Fill in any required parameters in the request body.
  4. Click "Send Request" to test the API.
  5. View the response in the right panel.
  6. Explore the interactive fleet map and dashboard for more insights.
